# 2020 Arduino Project

## 課程介紹

2020 年與師大創機(NTNUCMRC)舉行 Arduino 聯合社課的相關資料.

## 課程時間

* 目前一共 9 堂課.
* 03/12 起每周四, 18:30 ~ 21:00. 如放假將於FB專業公布.
